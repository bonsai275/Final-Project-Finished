# Melody Mix

## Description
This project is a web application that generates playlists based on different genres of music. It uses Flask as the web framework and interacts with a database to retrieve recent songs and generate playlists.

## Installation
1. Clone the repository to your local machine or download the zip file and extract it.
2. Install the required dependencies by running the following command in the project directory: pip install -r requirements.txt

## Usage
1. Make sure you have the database set up and running by using the command: python WebSite.py
2. Start the Flask application by running the following command in the project directory:
3. Open your web browser and navigate to `http://localhost:5000` to access the application.
4. The homepage will display the recent songs from the database as well as a embedded video of the most recent top song!
5. To generate a random playlist, click on the "Random Playlist" button.
6. To generate a playlist based on a specific genre, select the genre from the list of available genre buttons.
